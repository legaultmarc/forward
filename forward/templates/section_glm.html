Generalized Linear Models

To analyze dichotomous traits, we used Generalized Linear Models (GLMs). The outcomes were modeled using a binomial distribution and the logit link function was used resulting in the following model:

\[ \mathbb{E}(Y)=\mu=g^{-1}(\bf{X}\beta) \]

Where Y is the outcome (e.g. disease), g is a link function, X is a design matrix containing genotypes using additive coding and β is a vector of paremeters.
